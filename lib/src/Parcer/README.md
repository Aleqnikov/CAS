# Модуль Парсер

> Сейчас он АБСОЛЮТНО И ПОЛНОСТЬЮ НАВАЙБКОЖЕН!!! создается как бы прототип, который В будующем нужно ЦЕЛИКОМ ПЕРЕПИСАТЬ!!!!!!!!!! Делаю так, потому что нету времени, но идею проверить свою хочутся)))


 Данный модуль представляет собой инструментарий для парсинга ввода в формате LateX и затем вычисление его при попощи СКА! и затем возвращяет результат
 в то же формуле латех. Как работает парсер? 
 - Представление формулы в виде последовательности токенов при помощи алгоритма Ахо-Корасик.
 -  Валидация строки на корректность.
-  Исправление токенов, финальная версия.
- Определение приоритета операций
- Построение дерева выполнения операций
- Вычисление результата.
- Вывод результата.
